# CubeEdge IT Solutions

Professional multi-page website for CubeEdge IT Solutions featuring CallBee flagship product showcase.

## Tech Stack
- **Framework**: [Next.js 15+](https://nextjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Typography**: [Poppins (Google Fonts)](https://fonts.google.com/specimen/Poppins)

## Features
- **Home**: Modern, interactive hero section and "About Us" summary.
- **About**: Company vision, mission, and technical storytelling.
- **Products**: Case study style showcase for CallBee.
- **Contact**: Integrated contact channels and official company info.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
